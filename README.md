# FlexiStrategy.ai

Future‑ready AI, data & cloud strategy. This is a fast, static marketing website for FlexiStrategy.ai — a consulting firm delivering end‑to‑end coverage across cloud, data, and AI: strategy, implementation, and vendor‑neutral advisory.

## Tech Stack

- Next.js (App Router, static generation)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Run Locally

```bash
npm install
npm run dev
# open http://localhost:3000
```

## Build

```bash
npm run build
npm start
```

## Deploy

- Push to `main` branch on GitHub (`srini-data/flexistrategy`).
- Vercel will build and deploy automatically if GitHub integration is configured.
- Alternatively, use Vercel CLI:

```bash
vercel --prod --yes
```

## Notes

- Purely static site: no backend, no database.
- Uses system/Next fonts and lightweight CSS animations for performance.
- Images are SVG/gradients; no licensed photos used.
